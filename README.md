- Mainnet: https://thegraph.com/explorer/subgraph/cryptninja/indexs_order
- Kovan: https://thegraph.com/explorer/subgraph/cryptninja/index_orders_kovan

### Install

```bash
yarn
```

### Deploy mainnet

```bash
yarn run deploy:mainnet
```

### Deploy kovan

```bash
yarn run deploy:kovan
```
